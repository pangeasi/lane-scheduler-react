# Lane Scheduler React

A flexible, drag-and-drop scheduler component library for React with full TypeScript support.

**Unit Agnostic Design**: This library doesn't make assumptions about what your slots represent. Whether you're scheduling time intervals, physical spaces, resource allocations, or any sequential system - the components work the same way.

## Overview

Lane Scheduler React provides a powerful and flexible scheduling interface that allows users to:

- **Drag & Drop** appointments within and between lanes
- **Resize** appointments by dragging their edges
- **Block** specific time slots
- **Lock** appointments to prevent modification
- **Customize** rendering with render props
- **Handle** touch events for mobile support

## Core Concepts

### Scheduler

The `Scheduler` component provides the context for drag & drop operations. It coordinates state between multiple lanes and handles cross-lane appointment movements.

### Lane

The `Lane` component represents a single scheduling timeline divided into slots. Each lane can contain multiple appointments and has its own blocked slots and configuration. Slots are unit-agnostic - they can represent time periods, physical spaces, resources, or any sequential system.

### Appointments

Appointments are the schedulable items that can be moved and resized. They have properties like:

- `startSlot` - The starting time slot
- `duration` - How many slots the appointment spans
- `title` - Display text
- `locked` - Whether the appointment can be modified
- `allowOverlap` - Whether OTHER appointments are allowed to overlap with this one (not whether this appointment can overlap with others)
- `onBlockedSlot` - Custom logic for blocked slot handling
